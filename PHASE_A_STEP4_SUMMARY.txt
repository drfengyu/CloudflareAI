## 📊 功能特性

### 设计令牌集成

```typescript
// 颜色令牌
CHART_COLORS = [
  "var(--chart-1)",  // oklch(...) 主色
  "var(--chart-2)",  // oklch(...) 次色
  "var(--chart-3)",  // oklch(...) 第三色
  "var(--chart-4)",  // oklch(...) 第四色
  "var(--chart-5)",  // oklch(...) 第五色
]

// 样式令牌
grid: { stroke: "var(--border)" }
axis: { stroke: "var(--muted-foreground)" }
tooltip: { backgroundColor: "var(--card)" }
```

### 通用 Props

所有图表组件支持：
- `data` - 泛型数据数组
- `dataKey` - 数据键（Y 轴）
- `xAxisKey` - X 轴键
- `height` - 高度（默认 250px）
- `yAxisLabel` - Y 轴标签
- `tooltipFormatter` - Tooltip 格式化
- `xAxisFormatter` - X 轴格式化
- `emptyMessage` - 空数据提示

### BarChart 特有

- `colors` - 自定义颜色数组（默认 CHART_COLORS）
- `barRadius` - 圆角（默认 [4,4,0,0]）

### LineChart / AreaChart 特有

- `color` - 线条/填充颜色（默认 var(--primary)）
- `strokeWidth` - 线宽（默认 2）

## 🎯 使用示例

### BarChart（柱状图）

```tsx
import { BarChart } from "@/components/charts"

<BarChart
  data={modelUsage}
  dataKey="credits"
  xAxisKey="model"
  yAxisLabel="Credits"
  tooltipFormatter={(value, name, item) => {
    return [`${Math.round(value)} cr`, `${item.payload.model}`]
  }}
/>
```

### LineChart（折线图）

```tsx
import { LineChart } from "@/components/charts"

<LineChart
  data={dailyUsage}
  dataKey="credits"
  xAxisKey="date"
  color="var(--primary)"
  xAxisFormatter={(value) => value.slice(5)} // "2026-06-13" -> "06-13"
/>
```

### AreaChart（面积图）

```tsx
import { AreaChart } from "@/components/charts"

<AreaChart
  data={trends}
  dataKey="value"
  xAxisKey="time"
  color="var(--chart-2)"
  strokeWidth={3}
/>
```

### ChartEmpty（空状态）

```tsx
import { ChartEmpty } from "@/components/charts"

{data.length === 0 && <ChartEmpty message="今日暂无数据" />}
```
