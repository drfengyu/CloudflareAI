2. ✅ API Key 查看解密失败
   • 问题: 点击眼睛图标提示"解密失败"

   • 修复: 配置环境变量并更新文档
   • Commit: 1bc837e

3. ✅ API Key 名称显示错误
   • 问题: "dptest" 显示为 "Deepseek"（渠道名）
   • 原因: Drizzle ORM leftJoin 字段映射 bug
   • 修复: 移除 leftJoin，改用手动映射
   • Commit: 3c28ea1
