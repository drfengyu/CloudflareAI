## 🎯 使用流程

### Step 1: 创建渠道

访问 `/admin/channels`，点击"创建渠道"：

1. **渠道名称**: Vercel AI Gateway
2. **渠道类型**: OpenAI 兼容 (通用)
3. **配置**:
   - Base URL: `https://ai-gateway.vercel.sh/v1`
   - API Key: 留空（免费网关）

### Step 2: 健康检查

点击"健康检查"按钮：
- ✅ 连接正常 → 继续
- ❌ 连接失败 → 检查 baseUrl

### Step 3: 同步模型

点击"同步模型"按钮：

**系统自动**:
1. 调用 `GET {baseUrl}/models`
2. 解析模型列表
3. 插入 `model_pricing` 表
4. 设置 `multiplier = 1.0`

**返回结果**:
```json
{
  "message": "同步完成：新增 15 个模型，共 15 个",
  "total": 15,
  "inserted": 15,
  "skipped": 0
}
```

### Step 4: 创建 API Key

在 `/keys` 页面：
1. 选择渠道: Vercel AI Gateway
2. 选择模型: meta-llama/Meta-Llama-3.1-8B-Instruct
3. 创建密钥

### Step 5: 调用 API

```bash
curl https://cloudai.fuwari.fun/v1/chat/completions \
  -H "Authorization: Bearer sk-cfai-xxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "meta-llama/Meta-Llama-3.1-8B-Instruct",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
```

## 🔧 技术实现

### 适配器架构

```typescript
export class OpenAICompatibleAdapter implements ChannelAdapter {
  id = "openai-compatible"
  name = "OpenAI Compatible"
  type = "openai-compatible"

  // 转发到上游
  async route(path: string, request: Request, context) {
    const baseUrl = context.config.baseUrl
    const apiKey = context.config.apiKey
    const targetUrl = `${baseUrl}${path}`

    const headers = { "Content-Type": "application/json" }
    if (apiKey) headers.Authorization = `Bearer ${apiKey}`

    return fetch(targetUrl, { method, headers, body })
  }

  // 健康检查
  async healthCheck(context) {
    const res = await fetch(`${baseUrl}/models`, { headers })
    return { ok: res.ok, message: "..." }
  }

  // 获取模型
  async listModels(context) {
    const res = await fetch(`${baseUrl}/models`, { headers })
    const data = await res.json()
    return data.data.map(m => ({ id: m.id, object: m.object }))
  }
}
```

### 注册到系统

**文件**: `lib/channels/registry.ts`

```typescript
    key: "baseUrl",
    label: "Base URL",
    type: "text",
    required: true,
    placeholder: "https://api.xxx.com/v1 或 https://ai-gateway.vercel.sh/v1"
  }
]
```
