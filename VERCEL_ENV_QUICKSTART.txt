╚══════════════════════════════════════════════════════════════════════════════╝

📋 由于网络问题，请通过 Vercel Dashboard 手动配置

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🌐 步骤：

1. 访问：https://vercel.com/dashboard
2. 选择项目：CloudflareAI
3. Settings → Environment Variables
4. 点击 "Add New"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔑 新增环境变量：

┌─────────────────────────────────────────────────────────────────────────────┐
│ Name:                                                                       │
│   API_KEY_ENCRYPTION_SECRET                                                 │
│                                                                             │
│ Value (生产环境专用，与开发环境不同):                                       │

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ 配置后：

1. 保存环境变量
2. 触发重新部署（Deployments → Redeploy）
3. 等待部署完成
4. 访问生产站点测试

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🧪 测试验证：

1. 登录生产环境
2. 进入 /keys 页面
3. 创建新的 API Key
4. 点击眼睛图标 👁️
5. 确认能看到完整密钥（sk-cfai-...）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚠️  重要提醒：

• 生产环境密钥与开发环境不同（安全最佳实践）
• 需要在生产环境重新创建所有 API Key
• 旧密钥无法解密（使用了不同的加密密钥）
• 将生产密钥安全备份到密码管理器
